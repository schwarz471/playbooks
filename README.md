# My Ansible playbook

自分用の Ansible playbook

## available install

本ブランチ(osx)はMac OS X用
確認環境：OS X 10.11.3(El Capitan)
role指定によりインストール可能なソフトウェア(パッケージ)

- java7(jdk 1.7)
- git
- wget
- homebrew
  - homebrew-cask
- sublime text 3
  - Package Control
- pyenv
  - pyenv-virtualenv
- android-studio
